# 🚀 Codehelp Blogs UI

A modern, responsive blog card UI built using **HTML, CSS, and JavaScript**.
This project showcases a clean layout, interactive elements, and dynamic content rendering.

---

## 🌟 Features

* 🎨 Responsive Grid Layout (Mobile → Desktop)
* 🌙 Dark Mode Toggle
* 🖼️ Random Images using API (on every refresh)
* ⚡ Smooth Hover Animations
* 🧑 Author Section with Avatar
* 🏷️ Category Tags

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Flexbox + Grid)**
* **JavaScript**

---

## 📸 Preview

> A clean blog UI with cards, hover effects, and dark mode support.

---

## ⚙️ How It Works

### 🔹 Random Images

Images are loaded dynamically using:

```js
https://picsum.photos/
```

Each refresh fetches new images.

---

### 🔹 Dark Mode

* Toggle button adds/removes `dark` class on `<body>`
* CSS handles theme switching
